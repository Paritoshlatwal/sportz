## Featured Project

### [Sportz](https://github.com/Paritoshlatwal/sportz)

Sportz is a full-stack sports activity platform where users can create game posts, discover matches, send chat requests, and move into private conversations after requests are accepted.

**Highlights**

- Built with Next.js App Router, React, TypeScript, and MongoDB
- Credentials-based authentication with NextAuth
- Post creation, filters, notifications, and chat-request workflow
- Private chat between matched players after request acceptance
